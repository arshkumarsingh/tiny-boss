Tiny Boss Protocol

Cheap LLM reads the context. Smart LLM makes the decisions.

How It Works

  • • Supervisor analyzes task, asks 1 targeted question
  • • Worker reads full context, answers from context
  • • Supervisor synthesizes, asks follow-up or finishes
  • • Up to 3 rounds. Expensive model never sees full context

Cost Savings

  • • Groq worker: free tier (30 RPM, 500K tok/day)
  • • DeepSeek V4 Pro supervisor: $0.435/M input
  • • 12K-token paper: $0.0057 → $0.0011 (81% save)
  • • 20K-token log: $0.0089 → $0.0005 (94% save)

Best For

  • • Summarizing long documents / papers
  • • Extracting entities, dates, key facts
  • • "Find where X is discussed" searches
  • • Comparing multiple documents
